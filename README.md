# project-csc252
The SmartLibrary Project is a team-based programming assignment where students work in pairs to design and implement a real-world style application using core data structures and algorithms. 


Your solution must have:
• Dictionary/Map + Hashing (fast lookups by ISBN or title key),
• Balanced Tree or Heap/PQ (ordered listings, top-K, availability priority),
• Graph + Shortest Path (BFS for unweighted, Dijkstra for weighted).

Hash Table / Dictionary for O(1) average lookups by ISBN and/or title key.
• Balanced Search Tree (AVL or Red-Black) or a Min/Max-Heap for:
o ordered listings (tree) or availability prioritization (heap).
• Graph represented with adjacency list; implement:
o BFS (unweighted option),
o Dijkstra’s algorithm (weighted required).
• Queue/Stack as needed (BFS, path reconstruction).
• Time-complexity discussion for core ops in the report (see Deliverables).
